# 🛒 E-Commerce Backend API

A production-style E-Commerce Backend REST API built with Python, Django, and Django REST Framework. This project demonstrates modern backend development practices including JWT authentication, RESTful API design, user management, cart functionality, and order processing.

---

## 🚀 Features

- JWT Authentication (Login & Secure API Access)
- User Registration & Authentication
- Product Management
- Cart Management
- Order Management
- Django REST Framework Generic Views
- Serializer Validation
- Authentication & Permissions
- Modular Project Structure
- Clean REST API Architecture

---

## 🛠 Tech Stack

- Python
- Django
- Django REST Framework
- SQLite (Development)
- JWT Authentication
- Git & GitHub

## 🔐 Authentication

This project uses **JWT (JSON Web Token)** authentication to secure API endpoints.

Authenticated users can:

- Manage their cart
- Place orders
- Access protected resources

## 💻 Installation

```bash
git clone https://github.com/ZyntraTechX/E-commerce-Project.git

cd E-commerce-Project

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
